# Rishu Singh — Developer Portfolio

A single-page developer portfolio built with React, Vite, and Framer Motion.

Website Link:-https://rishu-portfolio-wheat.vercel.app/

## Stack

- React 19 + Vite
- Framer Motion (all animations)
- Plain CSS-in-JS (no Tailwind dependency)

## Features

- Cinematic mouse-reactive hero with ambient light beams and dust particles
- Terminal-voiced section dividers and a live `whoami` terminal card
- Bento-style skills grid
- Project cards with a mock-browser preview (no fragile iframe embeds)
- Magnetic scroll-spy navigation
- Command palette — press `Cmd+K` / `Ctrl+K` to jump anywhere on the site
- Custom cursor with trail effect
- Fully responsive, respects `prefers-reduced-motion`

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Deploy

This project deploys to Vercel with zero configuration — it's detected as a Vite app automatically.
